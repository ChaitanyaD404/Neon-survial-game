# Neon Survival Arena 

A futuristic browser survival shooter built using HTML5 Canvas, CSS3, and JavaScript.

## Features
- Smooth player controls
- Shooting mechanics
- Enemy waves
- Boss battles
- Powerups
- Shield + rapid fire
- Neon UI effects

## Controls
- WASD / Arrow Keys = Move
- Space = Shoot
